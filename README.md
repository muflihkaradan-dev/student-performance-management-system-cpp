# Student Performance Management System (C++)

Console-based student management system developed in C++.

## Features
- Tutor login and student login system
- Add, update, delete, and search student records
- Store and retrieve data using files
- View student progress card and GPA
- Sort students by ID, Name, or GPA

## Concepts Used
- Classes and Objects
- File handling
- STL (vector, algorithms)
- Menu-driven programming
- Password security using SHA-256

## How to Run
g++ *.cpp -o student -lssl -lcrypto 

./student
